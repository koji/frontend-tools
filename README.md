# FE-tools

## how to run

- clone this repo
- install packages
- run dev server

requirements: nodejs and yarn/npm  

```zsh
$ git clone https://github.com/koji/frontend-tools.git
$ cd frontend-tools

# yarn or npm for installing packages
$ yarn
$ npm i / npm install

# yarn or npm for running dev server
$ yarn dev
$ npm run dev
```

open http://localhost:3000 with a browser.
